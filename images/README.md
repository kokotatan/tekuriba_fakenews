# 画像ファイル管理

このフォルダには、Yahoo! JAPANページで使用する画像ファイルを格納します。

## フォルダ構成

```
images/
├── news/          # ニュース関連の画像
├── ads/           # 広告画像
├── logos/         # ロゴ画像
├── weather/       # 天気関連の画像
└── README.md      # このファイル
```

## 使用方法

1. 変更したい画像を適切なフォルダに配置
2. `yahoo.html` 内の画像パスを更新
3. ブラウザでリロードして確認

## 推奨画像サイズ

- **ニュース画像**: 120x80px
- **広告画像**: 300x250px
- **ロゴ**: 200x60px
- **天気アイコン**: 32x32px

## 対応フォーマット

- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- WebP (.webp)
